| 단계 | 역할 | 통과 조건 | 실패 처리 |

| Layer 2: 신호 트리거 | 진입 후보 생성 | 전략그룹별 점수 컷오프 충족 | HOLD |

## 2. 국면 분류 (5단계 + 보조 플래그)

### 2.1 입력 데이터 (모두 1H 캔들 기준)
| 입력 | 기간 | 계산 |
|------|------|------|
| EMA 20/50/200 | 각각 20/50/200봉 | 지수이동평균 (close) |
| ADX | 14봉 | Wilder smoothing (RMA) |
| +DI / -DI | 14봉 | Wilder 방향성 지표 |
| ATR | 14봉 | Wilder smoothing |
| ATR 20일 평균 | 480봉 (=20일) | SMA of ATR |
| 24H 가격변동률 | 24봉 | (close[-1] - close[-24]) / close[-24] |

### 2.2 판정 수식 (위에서 아래 순서, 첫 충족이 결정)

```python
# 1. CRISIS (최우선)
if ATR_now > ATR_20d_avg * 2.5 and price_change_24h < -0.10:
    regime = CRISIS

# 2. STRONG_TREND_UP
elif EMA20 > EMA50 > EMA200 and ADX > 25 and plus_DI > minus_DI:
    regime = STRONG_TREND_UP

# 3. WEAK_TREND_UP
elif EMA20 > EMA50 and ADX >= 20 and ADX <= 25 and plus_DI > minus_DI:
    regime = WEAK_TREND_UP

# 4. WEAK_TREND_DOWN
elif EMA20 < EMA50 and minus_DI > plus_DI:
    regime = WEAK_TREND_DOWN

# 5. RANGE (기본값)
else:
    regime = RANGE
```

### 2.3 보조 플래그 (메인 국면을 바꾸지 않음, 전략 제한만 추가)
- **RANGE_VOLATILE**: `ADX < 20 AND ATR_now > ATR_20d_avg × 1.2` (2봉 확인)
  - 효과: 반전포착(B) half size만, 브레이크아웃(C) Tier1/2만 허용
- **DOWN_ACCEL**: `EMA20 < EMA50 < EMA200 AND ADX > 22 AND -DI > +DI`
  - 효과: 반전포착(B) 최대 포지션 배수 ×0.4로 제한

### 2.4 히스테리시스 (국면 전환 안정화)
| 규칙 | 상세 |
|------|------|
| 전환 확인 | 새 국면 조건 3봉 연속 충족 시 전환 |

## 3. 전략 5종 상세

### 전략 A: 추세 추종 (Trend Following)
- **국면**: STRONG_UP, WEAK_UP
- **진입**: 1H EMA20 위에서 15M 풀백 → 15M EMA20 터치 후 반등 + RSI 40~60 + 거래량 평균 이상
- **청산**: 트레일링 스톱(ATR×2.5) + 부분청산(+3%→30%, +6%→30%, 나머지 트레일링)
- **목표**: 승률 40~50%, R:R 2.5:1 이상

> ⚠️ **비활성화됨** — OOS PF 0.55, 거래 부족. 데이터 축적 후 재검증 예정.

### 전략 B: 반전 포착 (Mean Reversion)
- **국면**: RANGE, WEAK_DOWN
- **진입**: RSI 30이하→35이상 반등 + BB 하단 이탈 후 복귀 + 거래량 2배 급증 (중 2개 이상)
- **청산**: BB 중간선 50% + 고정 손절 ATR×1.5
- **목표**: 승률 55~65%, R:R 1.2:1 이상

> ⚠️ **비활성화됨** — OOS 5건 거래, ADX 필터 적용 후에도 통계적 유의성 부족.

### 전략 C: 브레이크아웃 (Breakout)
- **국면**: RANGE에서 대기, 이탈 시 작동
- **진입**: 20봉 고점 돌파 + 돌파봉 거래량 > 평균 2배 + 1봉 확인 (거짓 돌파 필터)
- **청산**: 트레일링 ATR×3.0
- **목표**: 승률 35~45%, R:R 3:1 이상

> ⚠️ **비활성화됨** — STRONG_UP에서 RSI<30 조건 모순. 진입 시점 부족.

### 전략 D: 단기 스캘핑 (Scalping)
- **국면**: STRONG_UP + 고변동성
- **진입**: 15M RSI 과매도 바운스 + 상위 TF 추세 일치
- **청산**: 고정 TP 1.5%, SL 0.8%, 시간 제한 2시간
- **목표**: 승률 55~60%, R:R ~1.9:1

### 전략 E: DCA 매집 (Dollar-Cost Averaging)
- **국면**: CRISIS, WEAK_DOWN
- **대상**: Tier 1 코인(BTC, ETH)만
- **로직**: 하락장에서 정기적 소량 매수 → 상승 전환 시 추세추종으로 승격
- **사이징**: Core Pool의 4%/회 (고정)

## 7. 수수료 인식 필터
```
Expected Edge = 예상총이익률 - (수수료 + 예상슬리피지 + 실패페널티)
Expected Edge ≤ 0 → 진입 금지

수수료 = 0.50% (매수+매도)
슬리피지 = Tier별 추정 × 2 (매수+매도): T1 0.10%, T2 0.20%, T3 0.40%
```
