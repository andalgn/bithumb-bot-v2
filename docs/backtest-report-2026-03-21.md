# 백테스트 최적화 결과 리포트

**날짜**: 2026-03-21
**데이터**: 10개 코인 × 15M 3,000봉 / 1H ~5,000봉 (빗썸 API)
**수수료**: 0.25%/side + 슬리피지 0.1%/side = round-trip 0.70%

## 요약

| 지표 | Baseline (1차) | 최적화 후 (2차) |
|------|---------------|----------------|
| 활성 전략 | A/B/C/D/E (5개) | **A/E (2개)** |
| 전체 PF | 혼재 | A: **1.13** / E: 1.66 |
| 전체 승률 | 34% | A: **50%** / E: 60% |
| MDD (MC worst) | - | **7.0%** (< 15% 기준) |

## 전략별 상세

### Strategy A — 추세추종 (TREND_FOLLOW)

| 항목 | 1차 Baseline | 최적화 OOS |
|------|-------------|-----------|
| 파라미터 | sl_mult=1.0 tp_rr=3.5 | **sl_mult=3.0 tp_rr=2.0 cutoff=70** |
| 거래 수 | 318건 (full) | 140건 (OOS 30%) |
| 승률 | 27% | **50%** |
| PF | 0.44 | **1.13** |
| Expectancy | -0.0033 | **+0.0018** |

**개선 핵심**: SL을 1.0→3.0 ATR로 넓히고, TP를 3.5→2.0 R:R로 낮춤.
넓은 SL로 노이즈에 의한 조기 청산 방지, 낮은 TP로 실현 확률 증가.

**적용 국면**: STRONG_UP, WEAK_UP

### Strategy B — 반전포착 (MEAN_REVERSION) ❌ 비활성화

| 항목 | 1차 | 2차 (로직 개선 후) |
|------|-----|-------------------|
| Baseline PF | 0.26 | **0.68** (개선) |
| Best OOS PF | 0.52 | 0.55 |
| OOS 거래 | 15건 | 11건 |

**비활성화 사유**: RSI 진입 조건 완화(30→35)로 baseline 개선했으나, OOS에서 여전히 PF < 1.0.
거래 수도 부족하여 통계적 유의성 없음. 61% SL hit rate → 구조적 문제.

### Strategy C — 브레이크아웃 (BREAKOUT) ❌ 비활성화

| 항목 | 1차 | 2차 (ADX 필터 후) |
|------|-----|-------------------|
| Baseline PF | 0.50 | **0.81** (개선) |
| Best OOS PF | 0.82 | 5.00 (5건만) |
| OOS 거래 | 18건 | **5건** |

**비활성화 사유**: ADX 필터로 가짜 돌파 필터링 → baseline PF 0.50→0.81 상승.
그러나 OOS 5건만 남아 판단 불가. RANGE 국면에서 돌파 전략은 구조적 모순.

### Strategy D — 스캘핑 (SCALPING) ❌ 비활성화

**비활성화 사유**: STRONG_UP 국면에서 RSI < 30 반등 조건은 모순적.
전체 데이터에서 진입 시점 15건, IS 2건 → 과적합 불가피 (IS/OOS 괴리 400~676%).

### Strategy E — DCA 매집

| 항목 | 값 |
|------|-----|
| PF | 1.66 |
| 승률 | 60% |
| 대상 | BTC, ETH만 |
| 국면 | CRISIS, WEAK_DOWN |

**변경 없음**: 이미 수익 중. 파라미터 단순 (sl_pct=3%, tp_pct=5%).
