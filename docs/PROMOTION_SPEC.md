# 승격 시스템 명세

## 1. 승격 조건 (Active → Core)

진입 후 1~4시간 내에 아래 **모든 조건** 충족 시 승격:

| 조건 | 기준 |
|------|------|
| 수익 | +1.2% 이상 AND 2봉(2시간) 이상 유지 |
| 1H 추세 | 1H EMA20 위 + 1H 추세 방향 일치 |
| ADX | > 20 |
| 국면 | STRONG_UP 또는 WEAK_UP에서만 |
| Tier | Tier 3은 승격 불가 (Active에서만 거래) |

## 2. 승격 후 상태 전이

```
[Active 포지션] ─── 승격 조건 충족 ──→ [Core 보호기간]
                                           │
                    2봉 이내 손절선 이탈 ──→ [전량 청산]
                                           │
                    2봉 생존 ──────────→ [Core 정상]
                                           │
                    1H EMA20 이탈 ────→ [강등 → Active]
                    국면 악화         (재승격 전 6봉 검증)
                                           │
                    4봉 경과 + 재검증 ──→ [추가매수] (1회)
                                           │
                    트레일링/부분청산 ──→ [최종 청산]
```

## 3. 보호기간 규칙

| 항목 | 규칙 |
|------|------|
| 기간 | 승격 후 2봉(2시간) |
| 손절 | 이 기간 중 손절선 이탈 시 즉시 전량 청산 |
| 최대 허용 손실 | 원래 Active 사이즈 기준 손절폭으로 제한 |
| 의미 | 승격 직후 되돌림 방어 |

## 4. Core 정상 운영 규칙

| 항목 | 규칙 |
|------|------|
| 손절선 | 1H ATR × 2.5 (확대). 단 Active 손절폭의 2배 초과 불가 |
| 익절 | 제거 → 트레일링 스톱만 사용 |
| 청산 기준 | 1H 캔들 기준으로만 판단 (15M 무시) |
| 단순화 | 1H 양봉 지속 시 계속 보유 |
| 부분 청산 | +3% → 30%, +6% → 30%, 나머지 트레일링 |

## 5. 강등 조건 (Core → Active)

| 트리거 | 처리 |
|--------|------|
| 1H EMA20 하향 이탈 | Active로 강등, Active 규칙 적용 |
| 국면 악화 (예: WEAK_UP → RANGE) | Active로 강등 |
| 재승격 | 강등 후 최소 6봉 재검증 기간 |

## 6. 추가매수 조건

아래 **모든 조건** 충족 시에만 허용:
1. 승격 후 최소 4봉(4시간) 경과
2. 해당 전략 재평가 점수 ≥ 55점
3. 현재 수익 ≥ +2%
4. 최근 4봉 VWAP 위에서 집행 (평균단가 악화 방지)
5. 1H 추세 여전히 유지 (EMA20 위)
6. 해당 포지션 총 금액이 Core Pool의 25% 미만

| 항목 | 규칙 |
|------|------|
| 횟수 제한 | 포지션당 최대 1회 |
| 금액 | Tier 1: Core Pool의 15%, Tier 2: 8% |
