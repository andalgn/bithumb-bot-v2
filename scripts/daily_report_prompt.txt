You are a crypto trading bot analyst. Read the following SQLite database and generate a daily Discord-formatted report.

## Step 1: Read the database

The database is at `data/journal.db`. Read these tables using Python:

```python
import sqlite3, datetime
conn = sqlite3.connect("data/journal.db")
conn.row_factory = sqlite3.Row
cur = conn.cursor()

now = datetime.datetime.utcnow()
ts_24h_iso = (now - datetime.timedelta(hours=24)).isoformat()
ts_7d_iso  = (now - datetime.timedelta(days=7)).isoformat()
# entry_time / exit_time in trades table are Unix milliseconds
ts_24h_ms  = int((now - datetime.timedelta(hours=24)).timestamp() * 1000)
ts_7d_ms   = int((now - datetime.timedelta(days=7)).timestamp() * 1000)

# health_checks last 24h (created_at is ISO text)
cur.execute("SELECT * FROM health_checks WHERE created_at >= ? ORDER BY created_at DESC", (ts_24h_iso,))
health_rows = cur.fetchall()

# trades last 24h (entry_time is Unix ms)
cur.execute("SELECT * FROM trades WHERE entry_time >= ? ORDER BY entry_time DESC", (ts_24h_ms,))
trades_24h = cur.fetchall()

# trades last 7d (for strategy breakdown)
cur.execute("SELECT * FROM trades WHERE entry_time >= ? ORDER BY entry_time DESC", (ts_7d_ms,))
trades_7d = cur.fetchall()

# reflections last 7d (created_at is ISO text)
cur.execute("SELECT * FROM reflections WHERE created_at >= ? ORDER BY created_at DESC", (ts_7d_iso,))
reflections_7d = cur.fetchall()

conn.close()
```

Also check for recent log files: look in `data/` and `logs/` for `*.log` files modified in the last 24 hours (use Glob or Bash). Read up to 100 lines from the end of any found log files.
